# Signage HTML

Sistema simples de mídia indoor em HTML para exibição de vídeos, imagens e anúncios em tela cheia.

## Recursos
- Exibição automática de mídias
- Suporte para imagens e vídeos
- Layout responsivo
- Atualização rápida de conteúdo

## Como usar

1. Baixe os arquivos do projeto
2. Abra o arquivo `signage.html` no navegador
3. Adicione suas mídias na pasta do projeto
4. Atualize a lista de conteúdo conforme necessário
